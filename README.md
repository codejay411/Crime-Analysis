# Crime Analysis

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install library.

```bash
pip install virtualenv
```
```bash
virtualenv env_name
```
```bash
env_name/scripts/activate
```
## Start Project

Follow these command to start your project.

```bash
pip install -r requirements.txt
```
```bash
python app.py
```
